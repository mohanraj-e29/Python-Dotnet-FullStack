What is HTML?  HTML stands for Hyper Text Markup Language. HTML is the standard markup language for creating Web pages.
HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.
Question 3 
what is CSS?  Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML 
XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

What is java script?  JavaScript is a lightweight programming language that web developers commonly use to create more dynamic interactions when developing web pages
